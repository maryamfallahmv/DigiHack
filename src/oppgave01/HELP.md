# Hjelp — Oppgave 1: Hilsen til søker

Du skal returnere en streng som *inneholder* navnet, formatert akkurat
slik testen forventer.

- I TypeScript setter du sammen tekst med template literals (backticks)
  eller med `+`.
- Se på testen for å se nøyaktig hvordan resultat-strengen skal se ut —
  komma, utropstegn og mellomrom må stemme på tegnet.

Testen sammenligner strengen *eksakt* — ekstra mellomrom eller manglende
skilletegn gjør at den feiler.
